## 🎯 How It Works

### Filtering by App/Domain

1. **View Top Apps**: Look at "Top Domains/Apps" section
2. **Click an app**: Click on `chrome.exe` for example
3. **See filtered events**: Recent Events now shows only chrome.exe events
4. **See the badge**: "Filtered by: chrome.exe" appears above Recent Events
5. **Clear filter**: Click the badge or click the app again

### Example Workflow

```

🖥️ code.exe         87
```

## 🔐 Security Reminder

⚠️ **This version logs actual keystrokes!**
- All key presses are recorded in sequence
- Timestamps for each keystroke
- Visible in dashboard Recent Events
- Stored in SQLite database

**Ensure proper authorization and security:**
- Strong AUTH_KEY in config
- Secure network access
- Regular security audits
- User notification required
